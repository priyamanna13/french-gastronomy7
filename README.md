# French Gastronomy Website

## Overview

This project is a sophisticated, single-page website dedicated to French gastronomy. It's designed to be a visually rich and interactive experience, showcasing the art and tradition of French cuisine through elegant design and modern web technologies. The site features smooth page transitions, dynamic hero sections, a culinary gallery, and an interactive section on wine and cheese pairings.

## Features

* **Elegant Design**: A modern and clean aesthetic with a warm, earthy color palette and classic typography that captures the essence of French culinary tradition.
* **Smooth Page Transitions**: A custom full-screen overlay provides a seamless and polished navigation experience between pages.
* **Dynamic Hero Sections**: Each page begins with a unique hero section. The home page includes a subtle parallax effect and an automatically changing background image, while the pairing page features an animated 3D background created with **Three.js**.
* **Interactive Galleries**: The **Gallery** page showcases high-quality images with an interactive, custom cursor and hover effects that enhance user engagement.
* **Infinite Marquee**: An animated, continuous scroll of French food specialties and icons adds a dynamic visual element to the homepage.
* **Responsive Navigation**: A clean, hamburger-style menu is provided for mobile devices, offering a full-screen, animated menu overlay.
* **Interactive Pairing Cards**: The **Pairing** section features elegantly animated cards that highlight classic French wine and cheese combinations.

## Technologies

* **HTML5**: For the fundamental structure of the web pages.
* **CSS3**: Provides the styling, animations, and transitions.
* **Tailwind CSS**: A utility-first CSS framework used for rapid and efficient styling.
* **JavaScript (ES6+)**: Powers all interactive functionalities, including navigation, animations, and custom features.
* **Three.js**: A powerful 3D JavaScript library used to create the animated background on the Pairing page.
* **Font Awesome**: For the various food and navigation icons used throughout the site.
* **Google Fonts**: Used for the 'Playfair Display' and 'Poppins' fonts.
